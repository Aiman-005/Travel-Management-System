# ✈️ Travel Management System

A console based travel management application built in C as part of the Programming Fundamentals course at FAST-NUCES Karachi, our very first semester project.

## About

This system provides a platform for both travelers and managers to handle flight bookings, hotel reservations, and travel data all through a menu driven console interface with file based persistent storage.

## Features

- **Dual-role login** — separate portals for Traveler and Manager
- **Flight booking** — one-way and two-way flights with budget-based filtering
- **Hotel booking** — city and budget-based hotel selection
- **Manager portal** — full CRUD operations on flights, hotels, and bookings
- **File-based storage** — all customer and booking data persisted across sessions
- **Input validation** — email format validation, password confirmation, secure signup

## Tech Stack

- C (Procedural Programming)
- File handling for data persistence
- Menu driven interface with switch-case logic

## Manager Capabilities

| Function | Description |
|----------|-------------|
| View Bookings | Read all customer bookings from file |
| Update Hotels | Modify hotel listings per city |
| Update Airlines | Change ticket prices per airline |
| Remove Hotels | Delete hotel entries |
| Remove Airlines | Remove airline options |
